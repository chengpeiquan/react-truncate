## Out-of-the-box components

<CardGrid stagger>
  <Card title="ShowMore" icon="bars">
    Multi-line elliptical component with `expand` / `collapse` buttons, can be
    expanded with more custom functions.
  </Card>
  <Card title="MiddleTruncate" icon="add-document">
    More interesting than ordinary elision, positions can be specified to
    preserve the content of the first and last parts of the text, and only
    process the omitted components for the middle content.
  </Card>
  <Card title="Truncate" icon="pencil">
    The underlying basic components provide intelligent truncation function
    processing and can encapsulate more expansion capabilities according to your
    business needs.
  </Card>
